# Test Videos Directory

This directory is for test video files used in the CameraFeed component's test mode.

## Setup Instructions

To use the test videos, copy your video files here:

1. Copy `aag.mp4` from `C:\Users\Yashraj\Downloads\aag.mp4` to this directory
2. Copy `gira.mp4` from `C:\Users\Yashraj\Downloads\gira.mp4` to this directory

The files should be named:
- `aag.mp4`
- `gira.mp4`

## Usage

Once the files are in place, you can:
1. Enable "Test Mode" in the CameraFeed component
2. Select either `aag.mp4` or `gira.mp4` from the dropdown
3. Or use the "Custom File" input to select any video file

The videos will loop automatically for continuous testing of the threat detection model.
